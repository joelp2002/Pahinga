## 🎯 For Customers

### How to Make a Booking

1. **Visit the Homepage**
   - Browse available services
   - View pricing and amenities
   - Read service descriptions

2. **Click "Book Now"**
   - Located in the header or hero section

3. **Step 1: Select Service & Dates**
   - Choose service type (Pool, Cottage, or Event Hall)
   - Enter number of guests
   - Select check-in date
   - Select check-out date
   - View available services that match your criteria
   - Click on your preferred service card
   - Click "Continue to Your Information"

4. **Step 2: Enter Your Information**
   - Fill in your full name
   - Provide email address
   - Enter phone number
   - Click "Continue to Review"

5. **Step 3: Confirm Booking**
   - Review all booking details
   - Check dates, service, and total amount
   - Click "Confirm Booking"

6. **Receive Confirmation**
   - Get your booking reference number
   - Print or save the confirmation
   - Wait for staff to contact you within 24 hours

### Important Notes for Customers
- ✅ Bookings are FREE to submit
- ✅ Payment is arranged after confirmation
- ✅ Staff will contact you to confirm
- ✅ Keep your booking reference number
- ✅ Check-in times and policies will be communicated by staff

## 👔 For Employees

### Login Process

1. Click "Staff Login" on homepage
2. Enter credentials:
   - Username: `employee`
   - Password: `employee123`
3. You'll be redirected to Employee Dashboard

### Managing Bookings

#### View Bookings
- See all bookings in the main table
- Use the search bar to find specific bookings
- Filter by status using the dropdown

#### Update Booking Status
1. Find the booking in the table
2. Click the status dropdown in the "Update Status" column
3. Select new status:
   - **Pending**: Just received, not yet processed
   - **Confirmed**: Verified and approved
   - **Completed**: Service has been delivered
   - **Cancelled**: Booking was cancelled

#### Search Functionality
- Type in the search bar
- Search works for:
  - Customer name
  - Email address
  - Phone number
  - Booking ID
  - Service name

#### View Services
1. Click the "Services" tab
2. See all available services
3. Check availability status (green = available, red = unavailable)
4. View pricing and amenities

### Daily Tasks

**Morning Routine:**
1. Login to dashboard
2. Check "Today's Check-ins" counter
3. Confirm pending bookings
4. Contact customers if needed

**Throughout the Day:**
1. Monitor new bookings
2. Update statuses as services are delivered
3. Check service availability

**End of Day:**
1. Mark completed services
2. Review tomorrow's bookings
3. Ensure all statuses are current

## 👨‍💼 For Administrators

### Login Process

1. Click "Staff Login" on homepage
2. Enter credentials:
   - Username: `admin`
   - Password: `admin123`
3. You'll be redirected to Admin Dashboard

### Dashboard Overview

The admin dashboard has 3 main tabs:
1. **Bookings** - Manage all reservations
2. **Services** - Control availability
3. **Analytics** - View reports and statistics

### Managing Bookings

#### View All Bookings
- Complete table of all bookings
- Sortable columns
- Status indicators with color coding

#### Change Booking Status
1. Find the booking
2. Use the status dropdown
3. Select new status
4. Confirmation is automatic

#### Delete a Booking
1. Click the trash icon (🗑️) in the Actions column
2. Confirm deletion in popup dialog
3. Booking is permanently removed

⚠️ **Warning**: Deletion cannot be undone!

### Managing Services

#### Toggle Service Availability
1. Go to "Services" tab
2. Find the service you want to manage
3. Use the toggle switch on the right
4. Service availability updates instantly

**When to Disable a Service:**
- Maintenance required
- Fully booked
- Temporary closure
- Seasonal unavailability

#### View Service Details
Each service card shows:
- Service name and type
- Capacity (max guests)
- Price per day
- List of amenities
- Current availability status

### Analytics & Reports

#### Overview Statistics (Top Cards)
- **Total Bookings**: All-time booking count
- **Pending**: Bookings awaiting confirmation
- **Confirmed**: Active confirmed bookings
- **Total Revenue**: Sum of confirmed + completed bookings

#### Booking Status Chart (Pie Chart)
- Visual breakdown of all booking statuses
- Hover to see exact numbers
- Color-coded for easy reading

#### Revenue by Service (Bar Chart)
- Shows income generated per service
- Helps identify most profitable services
- Hover to see exact amounts

#### Service Utilization
- Progress bars showing booking frequency
- Identifies most popular services
- Helps with resource allocation

### Administrative Tasks

**Daily Tasks:**
1. Review new bookings
2. Confirm pending reservations
3. Check revenue statistics
4. Monitor service utilization

**Weekly Tasks:**
1. Review analytics trends
2. Adjust service availability as needed
3. Plan for upcoming busy periods
4. Clean up old completed bookings

**Monthly Tasks:**
1. Generate revenue reports
2. Analyze service performance
3. Plan maintenance schedules
4. Review pricing strategy

## 🆘 Troubleshooting

### Can't See Available Services?
- Check if dates are selected
- Verify dates are in the future
- Ensure service type is selected
- Try different dates if all services are booked

### Booking Won't Submit?
- Fill all required fields
- Check email format is correct
- Ensure dates are valid
- Verify service is selected

### Can't Login?
- Check username spelling
- Verify password (case-sensitive)
- Ensure you're using correct credentials for your role
- Contact system administrator if issues persist

### Dashboard Not Loading?
- Refresh the page
- Clear browser cache
- Try logging out and back in
- Check internet connection
